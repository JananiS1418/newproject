What is Pseudo class

A Pseudo class is used to define a special state of an element,it will give a extra attention to the users.This changes the background color of a button only when the mouse hovers over it.


Syntax for Pseudo class

selector:pseudoclass{ property:value; }


Usage of Pseudo class

hover - when mouse is over the element
active - when element is clicked
focus - when input is focused
visited - visited link color
first-child - first element of the parent
last-child - last element of the parent
nth-child(n)- selects specific child


When we use this for webpages

It is used in web pages to change the style or behaviour of elements based on their state,position, or interaction


List it all the webpages

The webpages are E-commerce site , Login/Signup forms, Educational portals

What is Pseudo Elements

A Pseudo element is used to style specific parts of an element,like the first letter or first line or inserting the content


Syntax for pesudo element

selector::pseudo_element{ property:value; }


Usage of Pseudo element

::first-letter - style the first letter
::firt-line - style first lines
::before - insert content before the element
::after - insert content after the element
::selection - style the selected text


when we use this for webpages

To add decorative content , to style specific text parts , to enchance user experience


List all the webpagse

Landing pages, portfolio websites , form pages, navigation bars
