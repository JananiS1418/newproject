What is JS?

JavaScript (JS) is a scripting language used to make web pages interactive, dynamic, and functional. It runs directly inside the web browser.

Why we use JS?

We use JavaScript to make webpages interactive and dynamic. It allows us to add animations, validate forms, update content without reloading, and respond to user actions like clicks or typing. While HTML builds the structure and CSS adds style, JavaScript controls the behavior, making websites more engaging and functional.

When we use JS ?

It is used to perform actions when users click buttons, fill forms, move the mouse, or type on the keyboard. JavaScript is also used to validate forms, update content without reloading, create animations, connect to servers, and control HTML and CSS

Where we use JS ?

We use JavaScript in many areas such as websites, web applications, mobile apps, and even servers. On websites, it is used to add interactivity and dynamic features. In web applications, JavaScript helps handle data and user actions in real time. If using JS what is happening on the webpage?

When we use JavaScript, the webpage becomes dynamic and interactive. Users can click, type, or drag elements, and the page responds instantly without reloading. It allows animations, sliders, dropdowns, and pop-ups to work smoothly. JavaScript also helps validate and submit forms smartly, making the website more user-friendly and responsive.

Any other frontend Programming languages like JS?

TypeScript - Superset of JS with static types; compiles to JS, safer and easier for large apps. Dart-Used with Flutter; can compile to JavaScript for web apps. CoffeeScript - A simpler syntax that compiles into JavaScript. Elm -Functional language that compiles to JS, used for reliable web interfaces. ClojureScript-Clojure language that compiles to JS, supports functional programming. Reason / ReScript-Strongly typed languages that compile to JS, often used with React.

Types of Using JS

1.Inline JavaScript - Written directly inside an HTML tag using the onclick, onchange, etc.
2.Internal JavaScript - Written inside the script tag within the html file
3.External JavaScript - Written in a separate .js file and linked to the HTML file
